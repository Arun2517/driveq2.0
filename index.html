Research Focus

What We're Building

DriveQ 2.0 targets key challenges in V2V communication — latency, reliability, and security at vehicular speeds.

DSRC & C-V2X Protocols

Comparing Dedicated Short-Range Communication with Cellular V2X for sub-10ms latency in dense traffic.

Communication Security

Lightweight cryptographic schemes for securing V2V message authenticity without latency overhead.

Edge Processing

On-device signal processing to reduce reliance on cloud infrastructure for time-critical decisions.

Collision Avoidance

Real-time V2V data fusion for cooperative collision warning and emergency braking systems.
